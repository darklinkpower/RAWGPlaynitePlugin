# RAWG Metadata plugin

A plugin for [Playnite](https://playnite.link/) which provides metadata from [RAWG](https://rawg.io/). This metadata includes:

- Tags (similar to Steam's user-generated tags)
- Genres
- Developers
- Publishers
- Links
- Release dates
- Metacritic ratings
